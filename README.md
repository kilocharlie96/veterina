# Veterinary Clinic Website

This is a **front-end project** for a veterinary clinic website, built with modern **HTML**, **modern CSS (CSS4 features)**, **JavaScript** and [HTMX](https://htmx.org/). The project focuses on clean code, responsive design, and contemporary web development practices.

## Technologies Used

- **HTML5**  
- **Modern CSS (CSS4)**
- **JavaScript ES6+**
- **HTMX**

## Features

- **Clean and semantic HTML**: Structured code for readability and maintainability.
- **Modern CSS**: Uses CSS Grid, Flexbox, clamp(), animations, variables, advanced selectors, and other modern techniques.
- **Progressive JavaScript integration**:
   -Adds interactive functionality (e.g., dynamic menu display in combination with HTMX).
   -Designed with scalability in mind — future features and enhancements will be built on this foundation.
- **HTMX integration**: Enables dynamic content loading and interactivity without heavy JavaScript.
- **Light/Dark mode support**: Automatically adapts to the user’s system theme.
- **Responsive design**: Fully responsive page that works seamlessly across devices.
- **Portfolio-ready**: Demonstrates best practices and a professional approach to front-end development.
